# Canna Studio

Canna Studio is a browser-based prototype for a 2D genre-focused vibe coding tool.

Open `index.html` in a browser to try it.

## Features

- Natural-language "Make" prompt for generating 2D game drafts
- 12 genre templates, including visual novel, roguelike, RPG, platformer, puzzle, card, rhythm, and life sim
- Game seed remixing across genres
- Feature atlas for game design, story, combat, accessibility, export, and workflow ideas
- 12 abstract visual themes inspired by glassy dark UI palettes
- Local JSON export/import and browser storage
